Preguntas

  1. ¿Qué es el DOM?

    2. Respuesta de ChatGPT:

    DOM (Document Object Model): Es una representación estructurada de un documento HTML o XML que permite acceder y manipular los elementos del documento.

    Fuente valida para esta definición

    DOM

  2. ¿Qué es un nodo?

    3. Respuesta de ChatGPT:

    Nodo: Es un objeto dentro del DOM que representa una parte del documento, como un elemento HTML, un atributo o un fragmento de texto.

    Fuente valida para esta definición

    Nodo

  3. ¿Cómo se accede a un elemento del DOM?

    4. Respuesta de ChatGPT:

    Acceso a un elemento del DOM: Se puede acceder a un elemento del DOM utilizando métodos como getElementById o querySelector, que permiten buscar y obtener referencias a elementos específicos.

    Fuente valida para esta definición

    Acceder a un elemento del DOM

  4. ¿Cómo se modifica un elemento del DOM?

    5. Respuesta de ChatGPT:

    Modificación de un elemento del DOM: Se puede modificar un elemento del DOM cambiando sus propiedades, como innerHTML para modificar su contenido o style para cambiar su estilo.

    Fuente valida para esta definición

    Modificar a un elemento del DOM

  5. ¿Cuál es la diferencia entre getElementby* y querySelector*?

    6. Respuesta de ChatGPT:

    Diferencia entre getElementby* y querySelector*: getElementby* busca elementos del DOM por atributos específicos como "id", "nombre de etiqueta" o "nombre de clase", mientras que querySelector utiliza selectores CSS para buscar elementos.

    Fuente valida para esta definición

    getElementby vs querySelector

  6. ¿Qué es un evento?

    7. Respuesta de ChatGPT:

    Evento: Es una acción que ocurre en el navegador, como un clic de ratón o la carga de una página.

    Fuente valida para esta definición

    Evento

  7. ¿Qué es un listener?

    8. Respuesta de ChatGPT:

    Listener: Es una función que se adjunta a un elemento del DOM para escuchar y responder a un evento específico.

    Fuente valida para esta definición

    Listener

  8. ¿Qué es un callback?

    9. Respuesta de ChatGPT:

    Callback: Es una función que se pasa como argumento a otra función y se invoca cuando se cumplen ciertas condiciones o eventos, como la ocurrencia de un evento.

    Fuente valida para esta definición

    Callback

  9. ¿Qué es un objeto de evento?

    10. Respuesta de ChatGPT:

    Objeto de evento: Es un objeto que se crea automáticamente y se pasa a un callback cuando se desencadena un evento. Contiene información sobre el evento, como su tipo, el elemento objetivo, etc.

    Fuente valida para esta definición

    Objeto de evento

Conclusiones

Experiencia general

Al utilizar estas herramientas he tenido una experiencia muy agradable en la mayoría de la actividad, ya que ahorra tiempo y son herramientas muy fáciles de utilizar. Sin embargo también veo que para obtener resultados óptimos y más adecuados se le debe especificar la instrucción de forma detallada y clara.

¿Cuándo y para qué usarías ChatGPT y cuándo Copilot?

ChatGPT lo usaría más para definiciones, conceptos, documentación y alguna verificación de algún código que quiera que me revise con las instrucciones adecuadas y precisas. Copilot lo usaría para cuando necesite autocompletar código, ver alternativas de alguna función que quiera implementar o editar algún pedazo de código.

Ventajas de usar AI para programar

Ayuda a poder realizar tareas más eficientemente y agiliza el trabajo.

Desventajas de usar AI para programar

Los programas pueden ser difíciles de interpretar y entender cómo llegan a sus decisiones. Otra desventaja es que a veces arroja código erróneo.
