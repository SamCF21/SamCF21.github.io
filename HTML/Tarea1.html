HTML cheat sheet

Etiquetas de HTML

  1. Etiqueta "a"

    2. a: Sirve para crear un enlace a una página web, archivos o rutas dentro de la misma página, direcciones de correo, o cualquier otra URL. Se debe de poner "href="..." " sefuido de la etiqueta ya que es la ruta a la que el link va a ir puesto.

    Ejemplo:

    Etiqueta a

  2. Etiqueta "p"

    3. p: Sirve para poder crear párrafos de texto

    Ejemplo:

    Un ejemplo de su uso es cuando se quiere dar a detalle algún contenido en una página o algún pedazo de texto que se quiere poner en particular como este.

  3. Etiquetas "h1, h2, ... , h6"

    4. h1, h2, ... , h6: Son títulos de contenido, cada uno es un subtítulo del anterior y según su número la letra es más chica.

    Ejemplo:

    HOLA

    HOLA

    HOLA

  4. Etiqueta "div"

    5. div: Sirve principalmente para contener a otros elementos HTML.

    Ejemplo:

    Este es un título en un div

    Esto es un texto que forma parte de un elemento div.

  5. Etiqueta "span"

    6. span: Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.

    Ejemplo:

    Ejemplo de span

    Esto es un párrafo dentro de un div, esto un span dentro de un párrafo.

  6. Etiqueta "strong"

    7. strong: Marca con énfasis las partes más importantes de un texto. En negritas.

    Ejemplo:

    Lo más importante en la vida del ser humano es la salud para poder realizar sus actividades cotidianas.

  7. Etiqueta "i"

    8. i: es un fragmento de texto que representa un cambio en el estado de ánimo o la calidad del texto. Este texto se muestra en cursiva.

    Ejemplo:

    Mis videojuegos favoritos son LEGO Star Wars TCS, Mario Kart Wii, Rocket League y Minecraft.

  8. Etiqueta "u"

    9. u: Se utiliza para subrayar texto.

    Ejemplo:

    Me encanta el helado de vainilla

  9. Etiqueta "b"

    10. b: Sirve para poner el texto en negritas. (bold)

    Ejemplo:

    Mi color favorito es el verde

  10. Etiqueta "strike"

    11. strike: Muestra el texto tachado con una linea horizontal.

    Ejemplo:

    Estimad@ ... A quién corresponda ...

  11. Etiqueta "small"

    12. small: Hace el tamaño del texto más pequeño

    Ejemplo:

    Hola mucho gusto! (Texto normal)
    Hola mucho gusto!! (Texto con small)

  12. Etiqueta "em"

    13. em: Marca con énfasis las partes importantes de un texto. En cursiva.

    Ejemplo:

    Esto es DEMASIADO IMPORTANTE para mí

  13. Etiqueta "header"

    14. header: Contiene al encabezado de una sección o documento, donde se colocan habitualmente los encabezados, vínculos de navegación, formularios de búsqueda, logos, tablas de contenidos, introducciones, entre otros elementos.

    Ejemplo:

    Inicio

    Tarea 1

    Tabla de contenidos

    • Etiqueta "a"
    • Etiqueta "p"
    • ...

  14. Etiqueta "section"

    15. section: Sirve para determinar qué contenido corresponde a qué parte de un esquema.

    Ejemplo:

    Encabezado

    Contenido que forma parte del encabezado.

  15. Etiqueta "article"

    16. article: Es un contenedor de bloques de contenido que se consideran independientes del sitio web y pueden, por lo tanto, ser vistos, reutilizados y distribuidos por separado.

    Ejemplo:

    El clima de hoy en CDMX

    4 de Abril de 2023

    Se pronostica un día soleado!

  16. Etiqueta "aside"

    17. aside: es un contenedor para información relacionada al documento pero no es estrictamente necesaria para su comprensión.

    Ejemplo:

    Tarea 1 sobre comandos para HTML
    English Version (Esta sección es realcionada al docs pero no crucial)

  17. Etiqueta "footer"

    18. footer: Es la sección de contenido que se encuentra en la parte inferior de una página web. Por lo general, contiene un aviso de derechos de autor, un enlace a la política de privacidad, un mapa del sitio, un logotipo, información de contacto, iconos de redes sociales y un formulario de registro por correo electrónico.

    Ejemplo:

    Copyright 2023© - Samantha Covarrubias - A01026174@tec.mx

  18. Etiqueta "nav"

    19. nav: Es una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos.

    Ejemplo:

    Link a tareas

  19. Etiqueta "img"

    20. img: Sirve para poner una imagen en el documento.

    Ejemplo:

  20. Etiqueta "br"

    21. br: Crea un salto de línea que es significativo para el contenido.

    Ejemplo:

    Este es un ejemplo
    del uso
    de la etiqueta
    br
    en html

  21. Etiqueta "hr"

    22. hr: Sirve para separar texto, mediante una línea horizontal. Representa un cambio de tema entre párrafos.

    Ejemplo:

    Este es un texto independiante.

    Este es un segundo texto, separado del primero.

  22. Etiqueta "form"

    23. form: Sirve para generar un formulario que contiene cajas de texto con casillas de verificación, botones y campos alfanuméricos. Funciona para recopilar datos de entrada del usuario con controles interactivos.

    Ejemplo:

    Formulario

  23. Etiqueta "table"

    24. table: Sirve para crear una tabla en el documento.

    Ejemplo:

    Tabla
    Elemento 1 Elemento 2 Elemento 3
    Contenido del
    Elemento 1    		 Contenido del
    Elemento 2    		 Contenido del
    Elemento 3

  24. Etiqueta "ul"

    25. ul: Se usa para definir una lista desordenada. (Lista con viñetas)

    Ejemplo:

    • Aquí
    • Acá
    • Allá

  25. Etiqueta "ol"

    26. ol: Se usa para definir una lista ordenada. (Lista numerada)

    Ejemplo:

    1. Aquí
    2. Acá
    3. Allá
