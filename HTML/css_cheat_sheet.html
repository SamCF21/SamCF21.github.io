Permite especificar el estilo, el ancho y el color del borde de un elemento.

Font-style establece si una fuente debe tener un estilo normal, cursiva u oblicua.

Font-weight establece qué tan gruesos o delgados deben mostrarse los caracteres en el texto.

9. Width, Height

"Width" y "Height" se utilizan para establecer el alto y el ancho de un elemento. Las propiedades de alto y ancho no incluyen "paddings", "borders" ni "margins". Establece el alto/ancho del área dentro del "padding", el "border" y el "marign" del elemento.

    Especifica el comportamiento de visualización (el tipo de cuadro de representación) de un elemento.

    Ejemplo:
